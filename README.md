# utmn-posters

Учебный веб-проект о рекламе еды в СССР 🍞
Сайт посвящен исследованию советской рекламы пищевой продукции: плакатам, визуальной агитации и связанным с этим историческим контекстом.

## О проекте

`utmn-posters` - это одностраничный сайт, собранный на `Gulp`, `Sass` и `Webpack`.  
Проект оформлен как интерактивная веб-страница с тематическими секциями, галереями плакатов, видео, графиком и дополнительными материалами в PDF.

В основе проекта лежит тема продовольственной рекламы в СССР: рыба, мясные изделия, хлебобулочная продукция, кукуруза и другие направления, отраженные в советской визуальной культуре.

## Возможности

- тематические контентные секции с историческими материалами;
- галереи плакатов со слайдерами;
- просмотр изображений через lightbox;
- видео-слайдер и модальное окно для просмотра видео;
- диаграмма на `Chart.js`;
- плавная прокрутка по странице;
- lazy loading изображений;
- сборка статических файлов для публикации.

## Технологии

Проект использует:

- `Gulp` для сборки и автоматизации;
- `Webpack` + `Babel` для обработки JavaScript;
- `Sass` для стилей;
- `BrowserSync` для локальной разработки;
- `Swiper` для слайдеров;
- `PhotoSwipe` для галереи изображений;
- `Chart.js` для диаграммы.

## Запуск проекта

Убедитесь, что у вас установлены `Node.js` и `npm`.

### Установка зависимостей

```bash
npm install
```

### Локальный запуск

```bash
npm run start
```

После запуска Gulp поднимет локальный сервер, соберет стили и скрипты, а также включит отслеживание изменений.

## Сборка проекта

```bash
npm run build
```

Команда создаст production-сборку в папке `dist`.

## Деплой

```bash
npm run deploy
```

В проекте настроен деплой через `gh-pages`, поэтому собранную версию можно публиковать как статический сайт.

## Структура проекта

```text
app/
  fonts/          # шрифты
  images/         # изображения и графические материалы
  js/             # исходный JavaScript
  parts/          # HTML-части и секции страницы
  public/         # дополнительные публичные файлы (PDF и др.)
  sass/           # стили на Sass
  index.html      # входная точка страницы

dist/             # итоговая сборка (генерируется автоматически)
gulpfile.js       # конфигурация сборки
package.json      # зависимости и npm-скрипты
```
